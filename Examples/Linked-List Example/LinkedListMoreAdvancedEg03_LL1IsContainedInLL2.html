In-class Example: Function to determine if the first of 2 given linked lists is contained in the second.
By definition:
List 1 is contained in List 2 if each distinct data item in List 1 occurs at least once in List 2.
An empty list is always contained in another list, even if the latter is empty.
Some implications of the definition:
List 2 can be shorter (have less number of nodes) than List 1 and still contain List 1. For instance, if List 1 has 1 million nodes all containing the same value, it contains only one distinct value. If List 2 contains only one node and the value of the data item in that node matches the only one distinct value contained in List 1, then List 1 is contained in List 2.
Code:
bool LL1IsContainedInLL2(Node* head1, Node* head2)
{
   if (head1 == 0) return true;  // empty list is always contained (even if 2nd list is empty)
   if (head2 == 0) return false; // list1 can't be empty so can't be contained if list2 is empty
   Node* copyOfHead2 = head2;
   while (head1 != 0)
   {
      while (copyOfHead2 != 0)
      {
         if (head1->data == copyOfHead2->data) break;
         copyOfHead2 = copyOfHead2->link;
      }
      if (copyOfHead2 == 0) return false;
      head1 = head1->link;
      copyOfHead2 = head2;
   }
   return true;
}
