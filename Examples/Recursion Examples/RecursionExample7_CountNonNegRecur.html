Recursion Example 7: CountNonNegRecur
Can you see that something's amiss in the following recursive function?
int CountNonNegRecur(const int a[], int n)
// n >= 1 assumed
{
   if (n == 1)
      if (a[0] >= 0)
         return 1;
      else
         return 0;
   else
      if (a[0] >= 0)
         return 1 + CountNonNegRecur(a + 1, n - 1);
      else
         CountNonNegRecur(a + 1, n - 1);
}
It illustrates a common error too many students tend to make.
