Recursion Example 3: FlipArray
Recursive solution:
// function to flip contents of given array a of size n
void FlipArrayRecur(int a[], int n)
{
   if (n <= 1) // nothing to do case (includes base case)
      return;
   else
   {
      FlipArrayRecur(a + 1, n - 2);
      int temp = a[0];
      a[0] = a[n - 1];
      a[n - 1] = temp;
   }
}

NOTE:
- Uses pointer arithmetic to get address of sub-array not including the first and last elements.
- Don't be misled into thinking that recursive calls always involve incrementing/decrementing something.

Compare it with iterative solution:
// function to flip contents of given array a of size n
void FlipArrayIter(int a[], int n)
{
   if (n <= 1) // nothing to do case (includes base case)
      return;
   else
   {
      int temp;
      for (int i = 0, j = n - 1; i < j; ++i, --j)
      {
         temp = a[i];
         a[i] = a[j];
         a[j] = temp;
      }
   }
}
