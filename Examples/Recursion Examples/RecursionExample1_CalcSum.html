Recursion Example 1: CalcSum
Recursive solution:
// function to calculate sum from 1 to given value n
// assume sum will not exceed range of int (n not too big)
int CalcSumRecur(int n)
{
   if (n < 1) // error case, indicated by 0 return value
      return 0;
   else if (n == 1) // base case
      return 1;
   else
      return n + CalcSumRecur(n - 1);
}
NOTE: Uses invocation construct (branching, i.e., function calling)
Compare it with non-recursive solutions:
Uses iteration/repetition construct (looping):
// function to calculate sum from 1 to given value n
// assume sum will not exceed range of int (n not too big)
int CalcSumIter(int n)
{
   if (n < 1) // invalid input, indicated by 0 return value
      return 0;
   int sum = 0;
   for (int i = 1; i <= n; ++i)
      sum += i;
   return sum;
}
Uses formula:
// function to calculate sum from 1 to given value n
// assume sum will not exceed range of int (n not too big)
int CalcSumFormula(int n)
{
   if (n < 1) // invalid input, indicated by 0 return value
      return 0;
   return n * (n + 1) / 2;
}
NOTE: Different algorithms for solving the same problem, with differing efficiencies.
