Recursion Example 6: FindMinIndexRecur
Returns the index rather than the value:
int FindMinIndexRecur(const int a[], int n)
// n >= 1 assumed
// in case of tie, returns lowest index of tied bunch
{
   if (n == 1)
      return 0;
   int minIndexTheRest = FindMinIndexRecur(a + 1, n - 1);
   if (a[0] <= a[minIndexTheRest + 1])
      return 0;
   else
      return 1 + minIndexTheRest;
}

QUIZ: What if, in case of tie, highest index of tied bunch needs to be returned?

Compare it with the case of returning the value (part of Recursion Example 5):
int FindMinRecur1a(const int a[], int n)
// n >= 1 assumed
{
   if (n == 1)
      return a[0];
   int minArrayMinusFirst =
       FindMinRecur1a(a + 1, n - 1);
   if (a[0] < minArrayMinusFirst)
      return a[0];
   else
      return minArrayMinusFirst;
}
For completeness, here's the "keep-last-element" (no-need-for-pointer-arithmetic) version:
int FindMinIndexRecur1(const int a[], int n)
// n >= 1 assumed
// in case of tie, returns lowest index of tied bunch
{
   if (n == 1)
      return 0;
   int minIndexTheRest = FindMinIndexRecur1(a, n - 1);
   if (a[n - 1] < a[minIndexTheRest])
      return n - 1;
   else
      return minIndexTheRest;
}
