Sample Past Final Exam Questions
Here are some sample past final exam questions aimed at giving you some idea with respect to the type and level of difficulty on the questions that I wrote in the past. You should not, in general, expect the questions that you will see in your exam to be identical in type, format, etc. In particular, you should not think that all you need to do to prepare for the exam is simply to learn how to do these sample questions.

What will be output if the following function is called given a pointer to the root of the original tree? (Use the definition for binary_tree_node given in the textbook and homework assignments.)  [ Ans ]
template <class Item>
void PrintSomething(const binary_tree_node<Item>* node_ptr)
{
   if (node_ptr != 0)
   {
      PrintSomething( node_ptr->left() );
      if ( node_ptr->left() == 0 && node_ptr->right() == 0 )
         cout << node_ptr->data() << ' ';
      PrintSomething( node_ptr->right() );
   }
}
Write a recursive function to compute the depth of a binary tree. Use the definition of depth given in the textbook: the root of a (non-empty) tree is at depth 0; the children of the root are at depth 1; the children of a node at depth d are at depth d+1; an empty tree has depth -1; the depth of a non-empty tree is the maximum depth of any node in the tree. The prototype and the precondition/postcondition contract for the function have been written for you.  [ Ans ]
template <class Item>
int depth(binary_tree_node<Item>* root_ptr);
// Precondition:  root_ptr is a pointer to the root of a binary tree.
//   The tree may be empty or non-empty.
// Postcondition: The depth of the binary tree is returned.
